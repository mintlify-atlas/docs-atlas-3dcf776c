## List Public Modpacks

<Api method="GET" url="/modpacks/public" />

List all public modpacks with pagination, search, and sorting capabilities. No authentication required.

### Query Parameters

<ParamField query="page" type="number" default="1">
  Page number for pagination (must be positive integer)
</ParamField>

<ParamField query="limit" type="number" default="10">
  Number of items per page (max 100)
</ParamField>

<ParamField query="search" type="string">
  Search term to filter modpacks by name or description
</ParamField>

<ParamField query="sortBy" type="string" default="createdAt">
  Field to sort by: `createdAt`, `updatedAt`, or `name`
</ParamField>

<ParamField query="sortOrder" type="string" default="desc">
  Sort order: `asc` or `desc`
</ParamField>

### Response

<ResponseField name="data" type="array">
  <Expandable title="Modpack Object">
    <ResponseField name="id" type="string">
      Unique identifier (UUID)
    </ResponseField>
    <ResponseField name="name" type="string">
      Modpack name
    </ResponseField>
    <ResponseField name="description" type="string">
      Optional description
    </ResponseField>
    <ResponseField name="avatarUrl" type="string">
      Optional avatar image URL
    </ResponseField>
    <ResponseField name="steamUrl" type="string">
      Optional Steam Workshop collection URL
    </ResponseField>
    <ResponseField name="owner" type="string">
      Owner user ID (UUID)
    </ResponseField>
    <ResponseField name="isPublic" type="boolean">
      Whether the modpack is publicly visible
    </ResponseField>
    <ResponseField name="isVerified" type="boolean">
      Whether the modpack is verified
    </ResponseField>
    <ResponseField name="createdAt" type="string">
      ISO 8601 timestamp
    </ResponseField>
    <ResponseField name="updatedAt" type="string">
      ISO 8601 timestamp
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="pagination" type="object">
  <ResponseField name="page" type="number">
    Current page number
  </ResponseField>
  <ResponseField name="limit" type="number">
    Items per page
  </ResponseField>
  <ResponseField name="total" type="number">
    Total number of items
  </ResponseField>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.pzpacks.com/modpacks/public?page=1&limit=10&search=zombie&sortBy=name&sortOrder=asc"
```
</RequestExample>

<ResponseExample>
```json Success Response
{
  "data": [
    {
      "id": "123e4567-e89b-12d3-a456-426614174000",
      "name": "Zombie Survival Pack",
      "description": "Essential mods for zombie survival",
      "avatarUrl": "https://example.com/avatar.jpg",
      "steamUrl": "https://steamcommunity.com/sharedfiles/filedetails/?id=123456",
      "owner": "user-uuid-here",
      "isPublic": true,
      "isVerified": false,
      "createdAt": "2024-01-15T10:30:00Z",
      "updatedAt": "2024-01-16T14:20:00Z"
    }
  ],
  "pagination": {
    "page": 1,
```

## Create Modpack

<Api method="POST" url="/modpacks" auth="bearer" />

Create a new modpack. The authenticated user becomes the owner.

### Body Parameters

<ParamField body="name" type="string" required>
  Modpack name (3-50 characters)
</ParamField>

<ParamField body="description" type="string">
  Modpack description (max 1024 characters)
</ParamField>

<ParamField body="avatarUrl" type="string">
  Avatar image URL (must be valid URL)
</ParamField>

<ParamField body="isPublic" type="boolean" default="false">
  Whether the modpack is publicly visible
</ParamField>

<ParamField body="steamWorkshopUrl" type="string">
  Steam Workshop collection URL (must be from steamcommunity.com)
</ParamField>

### Response

<ResponseField name="id" type="string">
  Created modpack UUID
</ResponseField>

<ResponseField name="name" type="string">
  Modpack name
</ResponseField>

<ResponseField name="owner" type="string">
  Owner user ID
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.pzpacks.com/modpacks" \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My Survival Pack",
    "description": "Custom survival mods collection",
    "isPublic": false
  }'
```
</RequestExample>

<ResponseExample>
```json Success Response
{
  "id": "987fcdeb-51a2-43c1-9876-543210fedcba",
  "name": "My Survival Pack",
  "description": "Custom survival mods collection",
  "owner": "user-uuid-here",
  "isPublic": false,
  "isVerified": false,
  "isActive": true,
  "createdAt": "2024-01-17T09:15:00Z",
  "updatedAt": "2024-01-17T09:15:00Z"
}
```

```json Error Response
{
  "error": {
    "message": "Name must be at least 3 characters long"
  }
}
```
</ResponseExample>

## Update Modpack

<Api method="PATCH" url="/modpacks/:id" auth="bearer" />

Update modpack details. Only the owner can update the modpack.

### Path Parameters

<ParamField path="id" type="string" required>
  Modpack UUID
</ParamField>

### Body Parameters

<ParamField body="name" type="string">
  Modpack name (3-50 characters)
</ParamField>

<ParamField body="description" type="string">
  Modpack description (max 1024 characters)
</ParamField>

<ParamField body="avatarUrl" type="string">
  Avatar image URL
</ParamField>

<ParamField body="isPublic" type="boolean">
  Public visibility status
</ParamField>

<ParamField body="steamWorkshopUrl" type="string">
  Steam Workshop URL (must be from steamcommunity.com)
</ParamField>

<ParamField body="metadata" type="object">
  <Expandable title="Metadata Object">
    <ParamField body="modsOrder" type="array">
      Array of mod IDs in desired order
    </ParamField>
    <ParamField body="modConfig" type="object">
      Configuration per mod ID with selectedSteamModIds
    </ParamField>
  </Expandable>
</ParamField>

<RequestExample>
```bash cURL
curl -X PATCH "https://api.pzpacks.com/modpacks/123e4567-e89b-12d3-a456-426614174000" \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Updated Survival Pack",
    "isPublic": true
  }'
```
</RequestExample>
